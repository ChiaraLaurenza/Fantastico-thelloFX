# Othello Game - GUI

A Java-based Othello (Reversi) game with a graphical user interface.

## Features

- Interactive GUI for playing Othello
- Human vs AI gameplay
- Visual board representation

## Requirements

- Java JDK 8 or higher
- JavaFX (for GUI)

## How to Run

1. Compile the project using your IDE (IntelliJ IDEA recommended)
2. Run the `Main.java` file in the `src/main/` directory

## Project Structure

```
src/
├── assets/          # Game images and resources
├── gui/             # GUI components and controllers
├── logic/           # Game logic and AI
├── main/            # Main entry point
└── szte/mi/         # Game model classes
```
